# USPTO Trademark Clearance Search — "Laverna" (Stage 0.4)

**Date:** 2026-07-16. **Scope:** free pre-filing clearance for the word mark
**LAVERNA** in **Class 9** (downloadable software / apps) and **Class 42**
(SaaS / software-as-a-service). Per the IP report, a full paid comprehensive
search (federal + state + common-law + app stores) is still recommended before
filing, but this establishes the federal picture.

## Findings (federal USPTO / TESS)

1. **LAVERNA — Registered (U.S. Reg. No. 3,303,422; Serial 78/422,114)**
   - **Owner:** Mattel, Inc. (El Segundo, CA)
   - **Filed:** May 2004
   - **Class:** **28** — toys, games and playthings, namely dolls, doll
     clothing and doll accessories.
   - **Status:** Registered (live per public listings).
   - **Conflict assessment for Class 9/42:** Low *direct* conflict — Mattel's
     registration is in Class 28 (toys/dolls), not software. But it is a
     **live, famous-ish consumer mark owned by a major corporation**, so a
     broad-expansion argument or opposion risk is non-zero if Mattel ever
     diversifies into kids' software under the same name.

2. **"Laverna" — established open-source software name (common-law / foreign).**
   - `Laverna/laverna` on GitHub is a pre-existing **JavaScript note-taking
     app** (Markdown + encryption, Evernote alternative). This is the
     **highest-relevance conflict** for a software product named Laverna: even
     though it is not a US federal trademark, it is prior use in the exact
     class of goods (downloadable software) and could support a **likelihood-
     of-confusion refusal** or a third-party opposition, and it dilutes the
     distinctiveness of the name for SEO/brand discovery.

## Conclusion

The IP report's prediction is confirmed: **"Laverna" is conflicted in
software.** No identical live Class 9/42 federal registration was found in this
free search, but (a) a live Mattel Class-28 registration exists, and (b) a
well-known open-source app already owns the name in software. Building brand
equity on "Laverna" for the product carries real confusion/dilution risk.

## Recommendation (from IP report Stage 0 / Thresholds)

- **Do NOT file a trademark for "Laverna" yet.** The name is conflicted in
  software precisely as the report warned.
- **Seriously consider "Wintermore Housekeeping"** as the product/brand mark
  to protect instead — it is more distinctive, lower-conflict, and a better
  candidate for a Class 9/42 filing near monetization.
- Keep "Laverna" as the **internal project / code name** (fine for an
  Apache-2.0 open-source repo), but lead any commercial brand with the more
  defensible mark.
- Before any filing, run a **paid comprehensive clearance** (federal +
  state + common law + app stores + domain) for the chosen mark.
